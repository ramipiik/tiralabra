# TicTacToe
Course project for tiralabra course at Helsinki University:  
https://tiralabra.github.io/2021_p1/index

## Quick guide
**Start the application:**  
```
python3 main.py
```

**Run all unit tests:**  
```
./run_all_tests.sh
```

**Check test coverage:**  

 ```
 pip install coverage
 ```
 ```
 coverage run main.py
 ```  
 ```
 coverage report -m
 ```

**Apply automatic styling:**  

```
pip install black
```

```
. black
```
